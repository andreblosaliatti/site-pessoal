Projeto em destaque: MedicFlow

MedicFlow

O MedicFlow nasceu para resolver gargalos comuns em clínicas de pequeno e médio porte: agenda descentralizada, prontuários dispersos e pouca previsibilidade do fluxo de atendimento. Meu papel como André Blos Aliatti foi estruturar a base técnica full stack, com foco em backend Java para regras de negócio, APIs REST consistentes e persistência relacional segura.

Tecnologias principais: Java, Spring Boot, PostgreSQL e React. Além da implementação, defini organização em camadas para separar domínio, aplicação e infraestrutura. Isso facilita manutenção e evolução. O resultado é uma base que permite adicionar módulos sem reescrever o sistema central.
