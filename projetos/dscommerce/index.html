Visão geral

O DSCommerce é um projeto de aprendizado fullstack (DevSuperior) que simula um e-commerce com camadas bem definidas, integração entre frontend e backend e foco em padrões de API.

O que eu pratiquei

  • Arquitetura em camadas (Controller / Service / Repository)
  • API REST com DTOs e validações
  • Integração frontend (React) consumindo endpoints
  • Boas práticas: organização, consistência e clareza de código

Stack e tecnologias

Java Spring Boot JPA / Hibernate Banco relacional React TypeScript (opcional)

Destaques técnicos

  • Modelagem de entidades e relacionamento
  • Endpoints REST com contratos consistentes
  • Padrões de DTO para entrada/saída
  • Front consumindo API com componentes reutilizáveis

Próximos passos

  • Refinar autenticação e perfis de acesso
  • Melhorar cobertura de testes (unitários e integração)
  • Melhorar UX e páginas do frontend
