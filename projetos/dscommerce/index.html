CASE TÉCNICO • FULLSTACK

DSCommerce Projeto fullstack (DevSuperior) com backend Spring Boot, frontend React e API REST

Status: concluído (estudo) • Jan 2026

Fullstack Java Spring Boot React REST

Visão geral

O DSCommerce é um projeto de aprendizado fullstack desenvolvido no ecossistema DevSuperior, simulando um e-commerce completo com integração real entre frontend e backend.

Problema que resolve

  • Entender o fluxo completo de uma aplicação fullstack
  • Padronizar comunicação entre frontend e backend
  • Aplicar arquitetura limpa em projetos reais

Arquitetura

  • Backend em camadas (Controller / Service / Repository)
  • API REST com DTOs para entrada e saída
  • Persistência com JPA/Hibernate
  • Frontend React consumindo endpoints REST

O que eu pratiquei

  • Modelagem de entidades e relacionamentos
  • Integração frontend/backend
  • Organização de código e padrões REST
  • Componentização no frontend

Próximos passos

  • Melhorar autenticação e perfis de acesso
  • Adicionar testes unitários e de integração
  • Evoluir UX e navegação do frontend
